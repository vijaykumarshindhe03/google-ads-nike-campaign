# Google Ads Campaign Project – Nike Running Shoes

## 📌 Project Overview
This project was created as part of my internship to understand how Google Ads campaigns are designed and structured.

I created a Search Campaign for Nike Running Shoes using the Google Ads platform.

## 🎯 Objective
The objective of this campaign was to drive website traffic to the Nike running shoes webpage and understand how search ads work in real-world scenarios.

## ⚙️ Campaign Details
- Campaign Type: Search Campaign
- Objective: Website Traffic
- Target Location: India
- Language: English

## 🔑 Key Features
- Keyword targeting based on user search intent
- Responsive Search Ads with multiple headlines
- Use of callout and sitelink extensions
- Optimization score and ad strength analysis

## 📂 Files Included
- Final Report (Word/PDF)
- Campaign Screenshots

## 📚 Learning Outcome
This project helped me understand practical aspects of digital marketing, including keyword selection, ad copywriting, and campaign optimization.
